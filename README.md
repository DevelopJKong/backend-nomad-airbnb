# 터미널 실행

```shell
poetry shell
```

# 장고 실행

```shell
poetry run python manage.py runserver
```

# 장고 마이그레이션

```shell
poetry run python manage.py migrate
```

# 장고 startapp

```shell
poetry run python manage.py startapp houses
```

# 장고 마이그레이션s

```shell
poetry run python manage.py makemigrations
```
